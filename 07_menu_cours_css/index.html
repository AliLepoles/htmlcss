Aide mémoire des sélecteurs CSS déja vus

  • * sélectionne TOUS les élements de la page html
  • .ma-jolie-nav sélectionne tous les élements ayant le nom de class "ma-jolie-nav"
  • #mon-unique-id sélectionne tous les élements ayant me nom de mon id "mon-unique*id"
  • idv, article, aside sélectionne toutes les balises <div>,<article>et<aside>
  • nav a sélectionne les liens <a>unclus/imbriqué dans la balise <nav>
  • nav a:hover sélectionne tous les liens de la nav et active une animation lors du survol de la souris

Propriétés CSS utilisées dans cette page

Décrivez leur usage et leurs valeurs possibles

  1. background: couleur du fond de l'élément
  2. color: couleur de la police
  3. float: effet flottant
  4. font-family: police
  5. font-size: taille de la police
  6. height: hauteur de l'élément
  7. left: position a partir de la gauche
  8. line-height : hauteur de l'interligne
  9. margin: marges exterieures
  10. padding: marges interieures
  11. position: position de l'élément dans la page (on a seulement utilisé la valeur (fixed) pour le moment)
  12. text-align: allignement du texte
  13. text-decoration: changement des valeurs héritées (ici le souligné standard pour les liens est supprimé)
  14. top: position a partir du haut de la fenêtre
  15. transition: temps de transition lors d'une animation (ici sur les liens de la barre de menus)
  16. width: largeur de l'élément par rapport a la taille de la fenetre si on indique une valeur relative (%, em,...)
